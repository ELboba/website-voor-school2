Turning ideas into interactive reality

1

HTML staat voor HyperText Markup Language. Het wordt gebruikt om de basisstructuur van websites te maken, en het is zelfs ouder dan ik! De afgelopen twee weken ben ik bezig geweest met het leren van HTML en CSS om mijn eigen portfolio-website te bouwen. Van tevoren wist ik niet wat ik moest verwachten – ik had nog nooit zoiets gedaan. Maar het bleek makkelijker dan ik dacht, mede dankzij de enorme hoeveelheid tutorialvideo’s die online beschikbaar zijn.

2

CSS staat voor Cascading Style Sheets, en het wordt gebruikt om websites te stijlen en tot leven te brengen. Ik heb CSS samen met HTML geleerd, en als ik moest kiezen welke van de twee moeilijker is, zou ik zonder twijfel CSS zeggen. Er zijn zoveel opties, eigenschappen en verschillende manieren om hetzelfde resultaat te bereiken, dat het in het begin overweldigend kan aanvoelen. Maar CSS leren is absoluut de moeite waard – het verschil tussen een website met CSS en een zonder is als dag en nacht. Een gestileerde website ziet er professioneel, aantrekkelijk en levendig uit. Ik begrijp nu volledig waarom mensen zeggen dat CSS de ziel van een website is.

>
3

JavaScript is een programmeertaal die speciaal is gemaakt om websites interactief te maken. Waar HTML de structuur van een pagina bepaalt en CSS voor de vormgeving zorgt, voegt JavaScript beweging, logica en dynamiek toe. Dankzij JavaScript kun je bijvoorbeeld knoppen laten reageren wanneer je erop klikt, formulieren automatisch laten controleren, animaties maken of gegevens direct ophalen zonder de hele pagina opnieuw te laden. Het is dus het onderdeel dat een website echt “slim” en gebruiksvriendelijk maakt
